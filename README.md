# Game Overview
Boba Barista is 3D Shooter Game that requires the player to be quick with their hands in assembling orders and serving it to the right customers. With this year's Global Game Jam theme being "bubble," the game showcases the aesthetics of the mainstream drink Bubble Tea and the mechanics or famous Bubble Shooter Games.
